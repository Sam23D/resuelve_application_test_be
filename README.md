# ResuelveBe

To install the project make sure you have elixir ~> "1.7" installed, then run.
```elixir
    mix deps.get
    mix compile
```
To run the program
```
iex -S mix
```
```elixir
    iex> ResuelveBe.run
```

Output shows final response from the `/conta` endpoint, as well as total executio time and total requests made
